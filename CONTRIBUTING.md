# Contributing

Thanks for contributing to this Inkbird Gateway Home Assistant integration.

## Development setup

1. Clone the repository.
2. Run tests:

```bash
python3 -m unittest discover -s tests -v
```

3. Keep changes focused and include/adjust tests for behavior changes.

## Pull requests

- Describe the problem and solution clearly.
- Include test updates where relevant.
- Keep documentation in sync (`README.md`, `changelog.md`).
